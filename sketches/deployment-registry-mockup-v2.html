🔌 Webhook Auto-Record

Add this step to your CI/CD pipeline after deploy. Records automatically — no manual entry needed.

- name: Record deployment in Anjungan run: | curl -s -X POST https://anjungan.edsuwarna.id/api/v1/deployments/webhook \ -H "Authorization: Bearer ${{ secrets.ANJUNGAN_TOKEN }}" \ -H "Content-Type: application/json" \ -d '{ "name": "Anjungan Backend", "environment": "production", "repo": "edsuwarna/anjungan", "branch": "main", "commit_sha": "${{ github.sha }}", "image": "ghcr.io/edsuwarna/anjungan-backend:${{ github.sha }}", "server": "peladen-central", "url": "https://anjungan.edsuwarna.id", "status": "running" }'
✅ Works with: GitHub Actions ✅ Forgejo Actions ✅ Any CI/CD that can call curl
