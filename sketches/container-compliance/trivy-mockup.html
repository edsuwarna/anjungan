How Source Differentiation Works
Each scan record stores a source field: "github_action" or "live". Dashboard groups by source and highlights discrepancies. When viewing a container, the system automatically fetches the latest scan from each source and performs a diff: CVEs present in Live but not in CI/CD (new zero-days, base image drift) vs CVEs fixed in CI/CD but still present in Live (outdated deployment).
