How it works (Phase 2)
Container-level scanning uses Docker Engine API over SSH to inspect each container. Checks include:
• Image vulnerability scan (Trivy/Docker Scout)
• Privilege mode detection
• Root user check
• Host network mode
• Capability audit (CAP_SYS_ADMIN, etc.)
• Bind mount / host path exposure
• Read-only rootfs enforcement
• Seccomp/AppArmor profile check
• Resource limit enforcement
• Health check configuration
Requires Docker Engine API access on the target server.
