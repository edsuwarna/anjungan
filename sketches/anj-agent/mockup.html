📦 Install Anj-Agent
Choose your OS and follow the instructions below
1
Download Binary
Download the latest agent binary for your architecture
curl -L https://releases.anjungan.io/anj-agent/v1.0.0/linux/amd64/anj-agent \ -o /usr/local/bin/anj-agent chmod +x /usr/local/bin/anj-agent
2
Register Agent
Run the registration command with your server token
anj-agent register \ --server https://anjungan.company.com \ --token ag_reg_kJ7hG2sQ9x \ --hostname server-prod-01
3
Verify Installation
Check that the agent is running and connected
systemctl status anj-agent # Expected: active (running) — Connected to anjungan.company.com
1
Run Docker Container
Pull and run the agent in a Docker container
docker run -d \ --name anj-agent \ --restart unless-stopped \ -v /var/run/docker.sock:/var/run/docker.sock:rw \ -e ANJ_SERVER=https://anjungan.company.com \ -e ANJ_TOKEN=ag_reg_kJ7hG2sQ9x \ -e ANJ_HOSTNAME=server-prod-01 \ ghcr.io/edsuwarna/anj-agent:latest
2
Check Container Logs
Verify the agent is connected
docker logs anj-agent --tail 20 # Expected: "Connected to anjungan.company.com"
1
Download Binary
Download the Windows executable
# PowerShell (Run as Administrator) Invoke-WebRequest -Uri "https://releases.anjungan.io/anj-agent/v1.0.0/windows/amd64/anj-agent.exe" \ -OutFile "C:\Program Files\Anjungan\anj-agent.exe"
2
Register & Start Agent
Register with your server token
anj-agent.exe register \ --server https://anjungan.company.com \ --token ag_reg_kJ7hG2sQ9x \ --hostname server-prod-01
✅ Prerequisites Checklist
  • ✓ curl installed (for downloading binary)
  • ✓ systemd (Linux) or service manager
  • ✓ Internet access to releases.anjungan.io
  • ✗ Docker socket (optional, for container management)
  • ✓ Registration token from Anjungan admin
